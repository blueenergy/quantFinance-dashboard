# 策略执行分析历史页面

## 概述

我们创建了两个独立的分析历史页面，以区分不同类型的分析：

1. **AI分析回溯** (`AIAnalysisHistory.vue`) - 显示大语言模型对股票的分析历史
2. **策略执行分析** (`StrategyExecutionAnalysis.vue`) - 显示策略执行过程中的复杂分析

## 功能特性

### 策略执行分析页面功能
- 筛选器面板：支持按策略类型、股票代码、日期范围筛选
- 统计概览：显示总分析次数、有效分析、成功率等关键指标
- 分析记录表格：展示分析时间、策略、股票、分析类型等信息
- 详情模态框：显示分析的推理过程和决策依据

### AI分析回溯页面功能
- 显示AI模型对股票的分析历史记录
- 展示投资建议、风险等级、技术分析等信息
- 提供分析详情查看功能

## 命名规范

- `AIAnalysisHistory.vue` - AI驱动的股票分析历史
- `StrategyExecutionAnalysis.vue` - 策略执行过程中的分析历史
- 清晰区分两种不同类型的分析，避免混淆

## API接口

- `/analysis/strategy/analysis-detail` - 获取策略分析详情
- 其他相关分析API端点

## 使用场景

### 策略执行分析
- 用于监控策略的决策过程
- 分析策略在不同市场条件下的表现
- 优化策略参数和逻辑

### AI分析回溯
- 用于查看AI模型的历史分析
- 评估AI分析的准确性
- 追溯AI决策过程

## 前端集成

- 在App.vue中添加了两个新的标签页
- 使用异步组件加载以优化性能
- 统一的样式和交互设计

## 用户体验

- 符合用户偏好的紧凑设计
- 详细的分析过程展示
- 清晰的分类和筛选功能
- 响应式设计，支持不同屏幕尺寸
